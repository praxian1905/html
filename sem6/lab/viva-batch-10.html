Question 91 • Unit 1 (Basics of CAD)

Differentiate between a 'Window' and a 'Viewport' in CAD graphics.

Answer: A window is the virtual boundary defining a specific mathematical coordinate area in the CAD world space that the user wants to view. A viewport, however, is the actual physical rectangular display area on the computer monitor (usually measured in pixels) where the contents of that mathematical window are mapped and rendered for the user to see.

Question 92 • Unit 2 (Curves and Surfaces)

How does the 'degree' of a curve affect B-Splines compared to Bezier curves?

Answer: In a Bezier curve, the degree of the polynomial is strictly tied to the number of control points (Degree = N - 1). If you add more points to make a complex shape, the math becomes unmanageably heavy and globally affects the curve. In a B-Spline, the degree is completely independent of the number of control points, allowing designers to create massively complex curves using simple, local-control, low-degree polynomials.

Question 93 • Unit 3 (Solid Modelling)

Explain the concept of 'Euler operations' in Boundary Representation (B-rep).

Answer: Euler operations are low-level topological commands used by the CAD kernel to build or modify a solid model while guaranteeing it remains mathematically valid. Every time a feature (like a face or edge) is added or removed, Euler operations ensure that Euler's formula (Vertices - Edges + Faces = 2) is strictly satisfied, preventing internal holes or non-manifold errors.

Question 94 • Unit 4 (CAE)

Briefly describe the difference between the 'h-method' and 'p-method' in FEA mesh refinement.

Answer: To achieve accurate mesh convergence, the h-method refines the mesh by physically making the elements smaller (increasing the total number of elements, 'h') around high-stress concentrations. The p-method, however, leaves the physical element size unchanged but increases the polynomial degree ('p') of the shape functions inside the element to calculate highly accurate internal stress distributions without remeshing.

Question 95 • Unit 5 (CAM)

What is an 'ATC' and what is its significance in a CNC machining center?

Answer: ATC stands for Automatic Tool Changer. It is a robotic carousel or magazine integrated into a CNC machine that holds multiple different cutting tools. Its significance is that it allows the machine to rapidly and autonomously swap tools (e.g., switching from a center-drill to a tap to an endmill) mid-program without human intervention, drastically reducing production cycle times.

Question 96 • Unit 6 (Latest Technologies)

Explain the difference between a 'Digital Twin' and a standard '3D CAD Model'.

Answer: A 3D CAD model is a static, virtual, geometric representation of a part's theoretical design. A Digital Twin is a dynamic, living counterpart to a physical asset; it combines the 3D model with real-time IoT sensor telemetry, allowing it to simulate, predict, and reflect the actual operating conditions, wear, and mechanical performance of the physical machine on the factory floor.

Question 97 • Unit 1 (Basics of CAD)

What is the mathematical difference in how Translation and Rotation are performed in 2D coordinates?

Answer: In standard 2D cartesian coordinates, translation requires matrix addition (adding Delta X and Delta Y to the original points). Rotation and scaling, however, require matrix multiplication. This mathematical mismatch makes composite transformations difficult, which is exactly why CAD systems convert everything into 3D Homogeneous Coordinates—so all operations can be handled uniformly via continuous matrix multiplication.

Question 98 • Unit 3 (Solid Modelling)

What is 'Feature Recognition' and why is it important for CAD/CAM integration?

Answer: Feature Recognition is an algorithmic process where CAM software analyzes a neutral 3D solid model (like a STEP file) and automatically identifies manufacturable engineering features—such as holes, pockets, slots, and fillets. It is crucial because it allows the CAM software to automatically select the correct cutting tools and generate optimized G-code toolpaths without the machinist manually defining every boundary.

Question 99 • Unit 5 (CAM)

What is 'Absolute' vs 'Incremental' programming in CNC (G90 vs G91)?

Answer: In Absolute programming (G90), all tool coordinates are calculated from a single, fixed origin point (Part Zero) on the workpiece. In Incremental programming (G91), every new coordinate is calculated relative to the tool's current resting position, meaning the origin effectively moves to the end of every cut. Absolute is preferred for safety, as a single error in incremental shifts the entire remaining toolpath.

Question 100 • Unit 6 (Latest Technologies)

How does 'Additive Manufacturing' (3D Printing) complement traditional CNC machining in modern smart factories?

Answer: Traditional CNC is a subtractive process, removing material from a block, which limits the creation of complex internal cavities. Additive manufacturing builds parts layer-by-layer, excelling at impossibly complex, lightweight topological geometries. Modern hybrid factories use Additive to print complex near-net shapes, and then use CNC machining to subsequently mill the critical mating surfaces and holes to extremely tight, polished tolerances.
